# 📻 Radio Xero - Progressive Web App (PWA)

¡Bienvenido al repositorio oficial de **Radio Xero**! Esta es una aplicación web moderna, rápida y "instalable" diseñada para ofrecer la mejor experiencia de radio digital directamente en tu dispositivo móvil o PC.

## ✨ Características Principales

-   **📱 PWA (Progressive Web App)**: Se puede instalar en la pantalla de inicio de Android y iOS sin necesidad de una tienda de aplicaciones.
-   **🎨 Diseño Premium**: Interfaz moderna con efectos de cristal (**Glassmorphism**), degradados suaves y animaciones fluidas.
-   **🔴 Indicador En Vivo**: Notificación visual dinámica para mostrar que la transmisión está activa.
-   **🔗 Integración Social**: Accesos directos rápidos a WhatsApp (pedidos), Facebook e Instagram.
-   **📅 Programación y Info**: Secciones dedicadas para mostrar los horarios de los programas y la información de la emisora.
-   **🚀 Optimizado para Desktop**: Diseño inteligente que utiliza todo el ancho de la pantalla en computadoras.

## 🛠️ Tecnologías Utilizadas

-   **HTML5 & CSS3**: Estructura y diseño responsivo avanzado (CSS Grid & Flexbox).
-   **JavaScript**: Registro de Service Workers y lógica de la PWA.
-   **Google Fonts**: Tipografía profesional (Inter & Poppins).
-   **Font Awesome**: Iconografía moderna.
-   **Zeno.fm**: Motor de streaming e integración del reproductor.

## 🚀 Instalación y Despliegue

1.  **Clona el repositorio**:
    ```bash
    git clone https://github.com/jf2021070309/radio-xero.git
    ```
2.  **Sube los archivos a GitHub Pages**:
    - Ve a `Settings` -> `Pages` en tu repositorio de GitHub.
    - Selecciona la rama `main` y guarda.
3.  **Instala la App**:
    - Abre el link generado en tu móvil (Chrome en Android o Safari en iOS).
    - Selecciona **"Instalar aplicación"** o **"Agregar a inicio"**.

## ⚙️ Personalización

Para ajustar la información a tus necesidades, edita el archivo `index.html`:

-   **Redes Sociales**: Busca las líneas con los enlaces de `wa.me`, `facebook.com` e `instagram.com`.
-   **Colores**: Modifica las variables CSS en la sección `:root`:
    ```css
    --primary: #ED1C24; /* Rojo oficial */
    --secondary: #1A3B70; /* Azul marino oficial */
    ```
